# Hoplon | Desenvolvedor Web Backend

Solução apresentada para [Hoplon | Desenvolvedor Web Backend](challenge.md).

## Requisitos

- [PHP](https://www.php.net/) 7.3
- [Composer](https://getcomposer.org/) 1.9

## Configuração

```
$ composer install
```

## Execução

```
$ php index.php
```

## Testes

```
$ ./vendor/bin/phpunit --testdox tests
```
